Como funciona?

O projeto Palpitão faz a otimização de palpites com base em estatísticas dos resultados, para cada modalidade lotérica.

Esses palpites são distribuídos sem repetição de combinações de dezenas. Assim, muitos apostadores que apostarem com o Palpitão, serão premiados em alguma faixa de premiação.

Os palpites otimizados são fixos, como no Projeto Minha Mega Sena ou dinâmicos como nas otimizações extremas para cada concurso.
